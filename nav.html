Projects

Jeopardy for Educators

A full stack application built using VueJS on the front end and Node/Express on the back end with a PostgreSQL database, deployed using Heroku. A tool for teachers to create custom jeopardy boards for in-class use. Teachers can create their own custom prompts or search through a database of over 300k actual Jeopardy questions that was scraped from the web.

DogMatch.com

A full stack application built using VueJS on the front end and Node/Express on the back end with a MongoDB database, custom middleware, and an outside API. TDD development using Jest. A tool for people to post dogs they have available or find their next dog best friend. Allows for filtering based on many criteria including breed, gender, health, personality attributes, and more.

Lewis In Space

A front end application build using VueJS. A game in which Lewis the dog faces slime pirates, an asteroid, and a wampa on his journey home to planet Alchemy. He faces consequences for all of his choices along the way.
